# Stray HELP

## Domain
Social Impact

## Problem Statement

In Mumbai alone there are over 90,000 stray animals, but only 20–30 recognized animal welfare organizations.

Because there is no fast and structured reporting system, and thus as a result many animals don’t receive timely help.

As per our research the problems we identified are:

1) Fragmented and scattered rescue requests - difficult to manage  
2) Lack of a centralised platform for NGOs  
3) Fundraising for specialized surgeries

## Solution

Stray Help is an AI powered platform that connects citizens with NGOs to rescue injured stray animals.

The work flow is as follows:

1) User uploads an image to report an injured stray animal  
2) AI creates an injury report and notifies nearest NGOs  
3) Necessary help is provided and updated on dashboard via NGOs

## Tech Stack

Frontend: React.js, Tailwind CSS  
Backend: Python, Django
AI Model: YOLOv8,KNN
Database: sqlite 
APIs: Gemini Vision Api

## Scalability

Our solution addresses a problem that is highly visible in urban areas where stray populations and rescue requests are frequent.  
It can be easily expanded beyond individual cities globally.
